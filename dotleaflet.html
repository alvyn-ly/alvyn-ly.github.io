DoT Leaflet

What is it?



DoT Leaflet is an application built from a collection of powerful tools and resources that help increase most major functions of project and work order management that exists in the San Jose Department of Transportation (DoT).

Built primarily in JavaScript with a good amount of HTML/CSS manipulation, DoT Leaflet leverages jQuery, RequireJS, Google Maps API, Salesforce API and more to enable functionalities that the users and developers would use on a daily basis.

Much of the functionality revolves around project tracking and map visualization, allowing for functions like:

  • Travel Routing
  • Custom Layers + ESRI Layer Support
  • Marker Population by SOQL/SQL Query + Marker Customization
  • Google Geolocation Search Engine
  • Marker Data Input/Output
  • Data Filtering
  • Click and Drag Functionality
  • ...and much more!




Made for functionality and customizability, DoT Leaflet provides key functions that help extract data from user interactions with the map to allow the developers to use and manipulate the data for any purpose.

Designed with speed in mind, the application silently filters out irrelevant and unseen data outside of the viewable range, reducing memory load and increasing speed in high data-density environments.
